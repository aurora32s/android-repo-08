
### 🎉 Git Repository Manager Service
자신의 github 이슈와 notification을 관리해보세요.🤗
<br/>

### 📌 제공기능
1. 로그인한 사용자의 Github 이슈/알림 확인
2. Repository 검색
3. 로그인한 사용자의 Profile 정보
